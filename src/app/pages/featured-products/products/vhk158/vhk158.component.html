Product Overview

The VHK158 Evaluation Kit – Designed for Next-Generation Compute & Networking Applications

The VHK158 Evaluation Kit is a high-performance computing and networking platform designed to handle memory-bound and compute-intensive applications. With 32GB of high- bandwidth memory (HBM), 112G PAM4 transceivers, PCIe Gen5x8, and QSFP28/QSFP-DD connectors, this evaluation kit is built to support applications requiring extreme data throughput, low-latency communication, and scalable processing capabilities.
